Toggle paste mode with F4.

Binds for insert: --------------------- {{{1
jk - Exit insert mode
<Tab> - complete word
CTRL-s - Correct most recent spelling mistake
<Up> - Enter ONE normal mode action.
<Down> - Enter an expression, then press <enter> to insert.

Binds for normal: ---------------------- {{{1
F10 - Disable/enable spell check
leader>hs - Toggle search highlighting
<leader>sy - Toggle syntax highlighting
<leader>wc - View word count.
<leader>b - Open buffer list to choose a buffer to jump to. 
<leader>evf - Edit vimrc file in split
<leader>svf - Source the vimrc file to update
s - Preform split change operation.
Left/Right arrow keys - Switch tab
H,L - Rebound to do 'extreme' motions. Eg H=Move to start of line, L=Move to end of line
<Space> - Clear search highlighting.
<C-t> - Transpose two words. Put cursor on second word.

Binds for plugins normal ------------------------------- {{{1
ShowMarks:
<leader>mt : Toggles ShowMarks on and off.
<leader>mh : Hides an individual mark.
<leader>ma : Hides all marks in the current buffer.
<leader>mm : Places the next available mark.


Binds for visual: ---------------------- {{{1
s - Sort selection.
<Up, Down> - Move text up and down. 

User Commands: -------------------- {{{1
Scratch - Create a scratch window, for jotting down disposable text.
SearchOpen - Search all open files for a pattern.
Start - Open this file again for reference.
StripWhitespace - Remove extra whitespace at ends of lines and file.
ForceQuit - Forcibly quit vim, discarding all unsaved changes
